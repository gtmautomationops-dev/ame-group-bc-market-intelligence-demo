## GitHub Pages Upload

Upload the contents of this folder to the root of a GitHub repository.

Required structure:

- `index.html`
- `styles.css`
- `script.js`
- `data/ame-proof-app-data.js`
- `artifacts/ame-proof-daily-briefing.html`
- `artifacts/ame-proof-morning-digest.html`

Recommended GitHub Pages settings:

1. Repository `Settings`
2. `Pages`
3. `Deploy from a branch`
4. Branch: `main`
5. Folder: `/ (root)`

Then open:

- Main dashboard: `/`
- Leadership Brief: `/artifacts/ame-proof-daily-briefing.html`
- Morning Digest: `/artifacts/ame-proof-morning-digest.html`
